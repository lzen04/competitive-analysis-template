# 경쟁사 분석 템플릿

## 이 프로젝트의 목적
사용자가 제품 아이디어와 알고 있는 경쟁사 2~3개를 입력하면,
에이전트가 경쟁사를 추가 발굴하고 전략 보고서를 자동 생성합니다.

## 사용 방법
1. `data/input.md`에 제품 정보와 알고 있는 경쟁사를 입력
2. `/competitive-analysis` 실행
3. `output/competitive-analysis-report.md`에서 결과 확인

## 분석 원칙
- 사용자가 입력한 경쟁사 + 에이전트가 추가 발굴한 경쟁사를 모두 분석
- 데이터 기반 사실과 전략적 해석을 명확히 구분
- 신뢰도가 낮은 정보는 반드시 [미확인] 표시
- 최종 전략 판단은 사용자(PM)가 내린다

## 출력 파일
- `output/competitive-analysis-report.md`: 전체 분석 보고서
