# 경쟁사 분석 스킬

## 트리거
/competitive-analysis

## 설명
data/input.md에 제품 아이디어와 알고 있는 경쟁사를 입력하면,
에이전트가 자동으로 경쟁사를 추가 발굴하고 전략 분석 보고서를 생성합니다.

---

## 실행 프롬프트

@data/input.md 파일을 읽고 아래 순서대로 경쟁사 분석을 수행해줘.

### Step 1: 입력 정보 파악
- 사용자의 제품 아이디어, 타겟 고객, 핵심 가치 제안을 정리
- 사용자가 입력한 경쟁사 목록 확인

### Step 2: 경쟁사 추가 발굴
사용자가 입력한 경쟁사 외에, 아래 기준으로 추가 경쟁사를 3~5개 더 발굴해줘:
- 직접 경쟁사: 같은 문제를 같은 방식으로 해결하는 제품
- 간접 경쟁사: 같은 문제를 다른 방식으로 해결하는 제품
- 잠재 경쟁사: 현재는 다른 영역이지만 이동 가능성이 있는 제품
각 경쟁사를 웹에서 검색해서 실제 존재하는 서비스인지 확인해줘.

### Step 3: 경쟁사별 심층 분석
발굴된 모든 경쟁사(사용자 입력 + 추가 발굴)에 대해 아래 항목을 조사해줘:
- 핵심 기능 (상위 3~5개)
- 가격 플랜 (무료/유료 구조)
- 타겟 고객
- 포지셔닝 메시지 (어떤 가치를 강조하는가)
- 강점 / 약점
- 최근 주목할 만한 움직임 (신기능, 투자, 마케팅 등)

### Step 4: 비교 매트릭스 생성
모든 경쟁사를 한눈에 비교할 수 있는 표를 만들어줘.

### Step 5: 화이트스페이스 분석
- 경쟁사들이 모두 못 하거나 안 하는 영역
- 고객 리뷰/불만에서 반복되는 미충족 니즈
- 사용자의 제품이 들어갈 수 있는 포지션 제안

### Step 6: 전략 제언
- 이길 수 있는 경쟁사 (공략 우선순위)
- 피해야 할 정면충돌 구도
- 차별화 포인트 3가지 제안
- 단기(3개월)/중기(1년) 포지셔닝 로드맵

### 출력
분석 결과를 output/competitive-analysis-report.md 파일로 저장해줘.
파일 상단에 분석 날짜와 대상 제품명을 명시해줘.
